# Lambda-e-Stream
<p1> Aqui dou inicio a Lambda e Stream em meus projetos de Java, a ideia foi montar sistemas simples de poo implementando essas ferramentas em cenarios diferentes e vendo como elas se comportaram

<p2> 1 -> Loja e produtos são uma atividade de Stream que a intenção era aplicar um devido desconto em itens com quatidade reduzida. 

<p3> 2 -> Já AppConta e Transação é um exemplo prático de um sistema bancário onde usamos Lambda para filtrar uma lista de objetos do tipo. 

<p4> 3 -> Em Funcionarios e Main é um sistema que imprime no console o nome de cada funcionário e o seu salário e depois ordenar os funcionários pelo salário.
